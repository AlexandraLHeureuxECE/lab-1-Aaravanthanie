# Tic-Tac-Toe Game

A simple, web-based Tic-Tac-Toe game for two players on the same device, built with HTML, CSS, and JavaScript.

## How to Run Locally

1.  Clone this repository or download the files (`index.html`, `style.css`, `script.js`).
2.  Ensure all three files are in the same directory.
3.  Open the `index.html` file in your favorite web browser.

That's it! The game will launch, and you can start playing.
